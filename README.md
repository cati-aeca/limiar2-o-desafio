# LIMIAR2: Segundo Balão Estratosférico do CATI

## 📡 Sobre o Projeto

LIMIAR2 é o segundo balão estratosférico lançado pelo CATI (Centro de Apoio Tecnológico e Inovação). Este repositório contém toda a documentação, dados e informações sobre o lançamento.

## 🚀 Estrutura do Site

O site GitHub Pages foi estruturado com as seguintes secções:

### 1. **Notícias**
   - Notícia 1
   - Notícia 2

### 2. **Missão**
   - Descrição Geral
   - Objetivos Gerais

### 3. **Desenvolvimento**
   - Payload
   - Instrumentação
   - Sistema de Recuperação - GPS

### 4. **Testes**
   - Calibrações

### 5. **Dados**
   - Dados Brutos (ficheiro TXT)
   - Gráficos:
     - Pressão vs Tempo de Voo
     - Pressão vs Altitude
     - Temperatura vs Altitude

### 6. **Contacto**
   - Informações de contacto

## 📁 Estrutura de Ficheiros

```
limiar2-o-desafio/
├── index.html              # Página principal
├── styles.css              # Estilos CSS
├── README.md               # Este ficheiro
├── images/                 # Pasta com imagens
│   ├── leca-fc-logo.png   # Logo Leça FC SAD
│   ├── cati-logo.png      # Logo CATI
│   ├── escola-logo.png    # Logo Escola
```

## 🖼️ Logotipos

Para completar o site, é necessário adicionar os seguintes ficheiros na pasta `images/`:

1. **leca-fc-logo.png** - Logo do Leça Futebol Clube SAD
2. **cati-logo.png** - Logo do CATI
3. **escola-logo.png** - Logo da Escola

## 📊 Dados e Gráficos

Na pasta `data/`, deve incluir:

1. **dados.txt** - Ficheiro com dados brutos do lançamento
2. Na pasta `images/`, os seguintes gráficos:
   - grafico-pressao-tempo.png
   - grafico-pressao-altitude.png
   - grafico-temperatura-altitude.png

## 🔧 Como Usar

1. Clone ou faça fork deste repositório
2. Adicione os logotipos na pasta `images/`
3. Adicione os dados e gráficos nas pastas correspondentes
4. Preencha o conteúdo das secções com as informações do projeto
5. Ative o GitHub Pages nas definições do repositório (branch: `main`, pasta: `root`)

## 📝 Como Editar o Conteúdo

Edite o ficheiro `index.html` e substitua o texto entre `[...]` pelo conteúdo real:

- `[Conteúdo da Notícia 1]` → Adicione o texto da primeira notícia
- `[Conteúdo da Notícia 2]` → Adicione o texto da segunda notícia
- `[Descrição geral do projeto LIMIAR2]` → Descrição completa
- E assim sucessivamente...
